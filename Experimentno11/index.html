XML Data Structure - Student Records

📋 About This Experiment

This experiment demonstrates basic XML structure with student records. XML is used to store and display structured data with custom tags.

📁 View Output Files

This experiment uses XML with CSS styling. The data structure contains student information stored in a custom XML format.

📄 View XML Output

🎯 Key Concepts

  • XML Declaration: Specifies the XML version and encoding
  • XML Stylesheet: Links CSS styling to XML content
  • Root Element: <students> contains all student records
  • Custom Tags: Tags like <student>, <name>, <email> define data structure
  • Hierarchical Structure: Parent-child relationships organize data logically

📁 Files Used

  • student.xml - Main XML data file
  • students.css - CSS styling for XML display

✨ Learning Outcomes

After completing this experiment, you should understand:

  • Basic XML syntax and structure
  • How to create custom XML tags
  • XML declaration and prolog
  • Linking stylesheets to XML documents
  • Data representation in XML format
